### Projeto
- a Sistema de Registro de Notas de alunos de uma instituição de ensino de pequeno porte.
- Implementar operações CRUD integradas a uma interface gráfica, garantindo interação entre usuário, aplicação e banco de dados. 

# Ferramentas
- Python 3 
- PostgreSQL + psycopg2
- Tkinter
